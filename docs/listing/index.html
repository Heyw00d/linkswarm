Listing Swarm

Get your site listed on 50+ directories automatically.
We handle signups, verification, and submissions.

How It Works

1

Set Up Email

Create a subdomain like listings.yoursite.com with IMAP access. We use this to receive verification emails.

2

Register Your Site

Tell us your URL, pick categories, and provide IMAP credentials. We scrape your site for name, description, and assets.

3

We Submit Everywhere

Our automation creates accounts, verifies emails, and submits your listing to every matching directory.

4

Get Your Backlinks

Pull a list of all confirmed listings. Export credentials to take full control of any account.

API Reference

Get Categories

GET /v1/listing/categories

Response:
{
  "crypto": { "name": "Crypto & Web3", "description": "..." },
  "ai": { "name": "AI & ML", "description": "..." },
  ...
}

Preview Match

POST /v1/listing/preview
{
  "categories": ["crypto", "fintech"]
}

Response:
{
  "total": 35,
  "free": 28,
  "paid": 7,
  "avgDR": 42,
  "estimatedPrice": 99
}

Register Site

POST /v1/listing/register
X-Payment: x402 (USDC)

{
  "url": "https://yoursite.com",
  "categories": ["crypto", "fintech"],
  "email": "submit@listings.yoursite.com",
  "imapCredentials": {
    "host": "mail.yoursite.com",
    "port": 993,
    "user": "submit@listings.yoursite.com",
    "password": "your-password"
  },
  // Optional - we scrape if missing
  "name": "Your Site",
  "tagline": "Short description",
  "description": "Full description...",
  "logo": "https://yoursite.com/logo.png",
  "screenshots": ["https://yoursite.com/screen1.png"]
}

Response:
{
  "siteId": "ls_abc123",
  "matchedDirectories": 35,
  "status": "pending"
}

Get Backlinks

GET /v1/listing/{siteId}/backlinks

Response:

    { "url": "https://another.com/p/yoursite", "dr": 42, "status": "verified" }
  ],
  "total": 28
}
