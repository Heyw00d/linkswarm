üöÄ Quick Start

1. Get your API key

Sign up at linkswarm.ai to get your API key instantly.

2. Verify your domain

Add a TXT record or meta tag to prove ownership:

POST /api/v1/sites/verify
{
  "domain": "yoursite.com",
  "method": "dns"  // or "meta"
}

3. Discover link partners

Find semantically relevant sites for exchange:

GET /api/v1/discover?domain=yoursite.com&limit=10

// Returns sites matched by:
// - Content similarity
// - Category overlap  
// - Reputation score

4. Propose an exchange

POST /api/v1/exchanges
{
  "from_domain": "yoursite.com",
  "to_domain": "partner.com",
  "from_page": "/blog/relevant-post/",
  "proposed_anchor": "useful resource"
}

üì° API Endpoints

GET /api/v1/registry

List all sites in the network (public)

POST /api/v1/sites

Register a new site

GET /api/v1/discover

Find relevant link partners

POST /api/v1/exchanges

Propose a link exchange

GET /api/v1/exchanges

List your pending and active exchanges

PATCH /api/v1/exchanges/:id

Accept, reject, or mark exchange complete
